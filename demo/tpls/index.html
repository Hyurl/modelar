Modelar

An expressive Model with Promise functionality and Query constructor. 

npm install modelar --save

Fast

Modelar is tiny but powerful, it not only speeds up the program, but also makes developing more efficient.

Easy

Modelar provides a lot of features that let you handle data in a very easy way, you just need to focus on how to display your site.

Functional

Query Constructor, Model Inheritance and Associations, Promise Ability and Event Listeners, all mean to reduce your work.
